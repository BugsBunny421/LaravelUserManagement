# #Basic User Registration and Login with laravel 9

### It used all the Laravel in-built tools, Good Starting point for Beginners.

Clone Project

```shell
git clone https://github.com/BugsBunny421/LaravelUserManagement.git
```
Install

```shell
composer install
npm install
npm run dev
```
Migrate

```shell
php artisan migrate
```
serve

```shell
php artisan serve
```
